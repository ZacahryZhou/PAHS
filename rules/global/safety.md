# Global Safety Rules

- Do not expose API keys, tokens, passwords, cookies, or credentials.
- Do not commit secrets.
- Do not run destructive commands unless explicitly approved by the user.
- Do not call staging tools from production orchestration.
- Do not mark a task as completed without user approval.
- Builder tools require tests, sandbox execution, and manual approval before production use.
- Learner suggestions must be pending until explicitly approved.
